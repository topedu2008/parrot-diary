## 📦 方式一：Web 版部署（推荐新手）

### 1. Vercel 部署（最简单）⭐

**步骤**：

1. **访问 Vercel**
   - 网址：https://vercel.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击「New Project」
   - 选择 `parrot-diary` 仓库
   - Root Directory 设置为：`web`
   - 点击「Deploy」

3. **等待部署**
   - 约 1-2 分钟完成
   - 获得域名：`https://parrot-diary-xxx.vercel.app`

4. **绑定自定义域名（可选）**
   - 在 Vercel 设置中添加域名
   - 配置 DNS 记录
   - 自动 HTTPS 证书

**优点**：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN
- ✅ 自动部署（Git 推送后）

---

### 2. Netlify 部署

**步骤**：

1. **访问 Netlify**
   - 网址：https://netlify.com
   - 使用 GitHub 账号登录

2. **添加站点**
   - 点击「Add new site」
   - 选择「Import an existing project」
   - 连接 GitHub，选择仓库
   - Build command 留空
   - Publish directory：`web`

3. **部署**
   - 点击「Deploy site」
   - 获得域名：`https://parrot-diary-xxx.netlify.app`

**优点**：
- ✅ 免费额度充足
- ✅ 表单功能
- ✅ 一键回滚

---

### 3. GitHub Pages 部署

**步骤**：

1. **启用 Pages**
   - 进入仓库 Settings
   - 找到「Pages」
   - Source：Deploy from a branch
   - Branch：选择 `main`
   - Folder：`/web`
   - 点击「Save」

2. **访问站点**
   - 地址：`https://yourusername.github.io/parrot-diary/web/`

**优点**：
- ✅ 完全免费
- ✅ 与 GitHub 深度集成
- ✅ 支持自定义域名

**缺点**：
- ❌ 国内访问慢
- ❌ 功能相对简单

## 📱 方式二：微信小程序部署

### 1. 准备工作

**注册小程序账号**：
1. 访问：https://mp.weixin.qq.com/
2. 点击「立即注册」
3. 选择账号类型：小程序
4. 填写信息并验证

**费用**：
- 个人主体：免费
- 企业主体：300 元/年（认证费）

---

### 2. 开发工具

**下载微信开发者工具**：
- 网址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 选择稳定版
- 安装后登录

---

### 3. 导入项目

1. **打开开发者工具**
2. **导入项目**
   - 选择目录：`parrot-diary/miniprogram/`
   - 填写 AppID（测试可用「测试号」）
   - 点击「导入」

3. **编译预览**
   - 点击「编译」按钮
   - 在模拟器中查看效果
   - 点击「预览」在真机测试

---

### 4. 代码上传

1. **点击「上传」按钮**
2. **填写版本信息**
   - 版本号：1.0.0
   - 项目备注：初始版本
3. **确认上传**

---

### 5. 提交审核

1. **登录小程序后台**
   - https://mp.weixin.qq.com/

2. **版本管理**
   - 找到刚上传的版本
   - 点击「提交审核」

3. **填写审核资料**
   - 功能截图
   - 隐私政策链接
   - 用户协议链接
   - 测试账号（如需登录）

4. **等待审核**
   - 通常 1-3 个工作日
   - 审核结果会短信通知

---

### 6. 发布上线

1. **审核通过后**
   - 登录小程序后台
   - 找到「审核版本」
   - 点击「发布」

2. **用户可见**
   - 发布后即可搜索到
   - 可分享到朋友圈

---

## 🐛 常见问题

### Q1: Web 版部署后访问 404？

**A**: 检查 vercel.json 配置，确保路由正确。

### Q2: 小程序审核被拒？

**A**: 常见原因：
- 功能不完整
- 隐私政策缺失
- 内容违规
- 类目选择错误

**解决**：根据审核反馈修改后重新提交。

### Q3: 图片上传失败？

**A**: 检查：
- 是否配置上传域名
- 文件大小是否超限
- 存储空间是否足够

### Q4: 数据丢失？

**A**: 本地存储版本：
- 清除缓存会丢失数据
- 不同设备数据不互通
- 建议添加导出功能
