## 📋 前置准备

### 1. 注册微信小程序账号

**网址**: https://mp.weixin.qq.com/

**所需材料**:
- 个人：身份证、手机号
- 企业：营业执照、对公账户

**费用**: 
- 个人主体：免费
- 企业主体：300 元/年（认证费）

### 2. 下载微信开发者工具

**下载地址**: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

**选择版本**: 
- Windows / Mac / Linux
- 稳定版（推荐）或  nightly 版

1. 登录微信小程序后台
2. 进入「开发」→「开发管理」→「开发设置」
3. 复制 **开发者 ID**（AppID）

## 🛠️ 本地开发

### 1. 克隆项目

```bash

```

### 2. 导入项目

1. 打开微信开发者工具
2. 点击「+」或「导入项目」
3. 选择项目目录：`parrot-diary/miniprogram/`
4. 填写 AppID（测试可用「测试号」）
5. 点击「导入」

## 💡 开发技巧

### 1. 本地存储使用

```javascript
import storage from '../../utils/storage';

// 添加鹦鹉
const parrot = storage.addParrot({
  name: '小灰',
  breed: '玄凤鹦鹉',
  birthday: '2025-01-15',
  gender: 'female'
});

// 添加照片
storage.addPhoto(parrot.id, {
  type: 'photo',
  path: filePath,
  caption: '学唱歌',
  tags: ['第一次']
});

// 获取数据
const photos = storage.getParrotMedia(parrot.id);
```

### 2. 图片处理

```javascript
// 选择图片
wx.chooseImage({
  count: 9,
  sizeType: ['compressed'],
  sourceType: ['album', 'camera'],
  success: (res) => {
    const tempFilePaths = res.tempFilePaths;
    // 保存到用户目录
    const savedPath = storage.saveFile(tempFilePaths[0], 'albums/' + parrotId);
  }
});
```

### 3. 录音功能

```javascript
// 开始录音
const recorderManager = wx.getRecorderManager();

recorderManager.start({
  duration: 60000, // 最长 60 秒
  sampleRate: 44100,
  numberOfChannels: 1,
  encodeBitRate: 128000
});

// 停止录音
recorderManager.stop();

// 监听录音完成
recorderManager.onStop((res) => {
  const tempFilePath = res.tempFilePath;
  // 保存录音文件
  const savedPath = storage.saveFile(tempFilePath, 'trainings/' + parrotId);
});
```

### 4. 音频播放

```javascript
// 创建音频上下文
const audioContext = wx.createInnerAudioContext();

// 设置音频源
audioContext.src = '/assets/audios/signals/SIG001.mp3';

// 播放
audioContext.play();

// 暂停
audioContext.pause();

// 监听结束
audioContext.onEnded(() => {
  console.log('播放完成');
});
```

## 📊 数据存储限制

### 微信小程序限制

| 类型 | 限制 | 说明 |
|------|------|------|
| 本地缓存 | 10MB | 文本数据 |
| 文件系统 | 200MB | 图片、视频、音频 |
| 单次写入 | 1MB | 限制 |

### 优化建议

1. **图片压缩**: 压缩到 500KB 以内
2. **视频限制**: 最多 5 秒，自动压缩
3. **定期清理**: 提供清理缓存功能
4. **缩略图**: 列表页使用缩略图

## 🔧 常见问题

### Q1: 真机测试看不到图片？

**A**: 检查文件路径是否正确，使用 `wx.env.USER_DATA_PATH` 保存文件。

### Q2: 录音功能无法使用？

**A**: 需要在 app.json 中配置权限：
```json
"permission": {
  "scope.record": {
    "desc": "用于录制训练音频"
  }
}
```

### Q3: 存储空间不足？

**A**: 实现清理缓存功能，定期清理不需要的文件。

### Q4: 如何备份数据？

**A**: 使用 `storage.exportData()` 导出数据，保存到云盘或发送给用户。
