## 🚀 获取 APK 的三种方式

### 方式一：GitHub Actions 自动构建（推荐⭐）

**最简单，无需任何环境**

1. **访问 GitHub Actions**
   ```
   https://github.com/topedu2008/parrot-diary/actions
   ```

2. **找到最新构建或手动触发**
   - 自动触发：每次推送到 main 分支
   - 手动触发：点击 "Run workflow"

3. **下载 APK**
   - 等待构建完成（约 10-15 分钟）
   - 在 Artifacts 中下载 `app-debug.apk`

**优点**：
- ✅ 无需本地环境
- ✅ 自动化的可靠构建
- ✅ 每次提交自动构建

**预计时间**：15 分钟

---

### 方式二：本地电脑构建

**如果你有 Windows/Mac/Linux 电脑**

1. **安装环境**
   - Node.js 16+
   - Java JDK 17+
   - Android Studio

2. **克隆项目**
   ```bash
   git clone https://github.com/topedu2008/parrot-diary.git
   cd parrot-diary
   ```

3. **构建 APK**
   ```bash
   npm install
   npx cap sync android
   npx cap open android
   ```
   
   然后在 Android Studio 中：
   - Build → Build APK

**优点**：
- ✅ 完全控制
- ✅ 可以快速迭代
- ✅ 适合后续开发

**预计时间**：首次 1 小时（含环境安装），后续 5 分钟

---

### 方式三：Web 版本直接测试

**最快，无需等待**

1. **访问在线版本**
   ```
   https://parrot-diary.vercel.app
   ```

2. **或使用本地预览**
   ```bash
   cd /home/admin/openclaw/workspace/parrot-diary
   npx http-server web -p 8080
   ```

3. **手机访问**
   - 用手机浏览器打开网址
   - 添加到主屏幕（类似 APP）

**优点**：
- ✅ 立即可以使用
- ✅ 无需安装
- ✅ 跨平台

**缺点**：
- ❌ 不是真正的 APP
- ❌ 功能受限

**预计时间**：1 分钟
