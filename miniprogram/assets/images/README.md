# TabBar 图标说明

## 需要的图标文件

请将以下图标文件放入 `assets/images/` 目录：

### 图标列表（共 10 个）

| 文件名 | 尺寸 | 说明 |
|--------|------|------|
| `tab-home.png` | 81x81 | 首页图标（未选中） |
| `tab-home-active.png` | 81x81 | 首页图标（选中） |
| `tab-album.png` | 81x81 | 相册图标（未选中） |
| `tab-album-active.png` | 81x81 | 相册图标（选中） |
| `tab-guide.png` | 81x81 | 指南图标（未选中） |
| `tab-guide-active.png` | 81x81 | 指南图标（选中） |
| `tab-training.png` | 81x81 | 训练图标（未选中） |
| `tab-training-active.png` | 81x81 | 训练图标（选中） |
| `tab-diary.png` | 81x81 | 日记图标（未选中） |
| `tab-diary-active.png` | 81x81 | 日记图标（选中） |

## 图标要求

- **格式**: PNG
- **尺寸**: 81x81 像素
- **颜色**: 
  - 未选中：#999999（灰色）
  - 选中：#4CAF50（绿色）
- **背景**: 透明

## 临时解决方案（开发测试用）

如果暂时没有图标，可以：

1. 使用微信开发者工具默认图标
2. 或使用 emoji 表情生成简单图标
3. 或从 iconfont.cn 下载免费图标

## 推荐图标资源

- **阿里巴巴矢量图标库**: https://www.iconfont.cn/
- **IconPark**: https://iconpark.oceanengine.com/
- **Flaticon**: https://www.flaticon.com/

搜索关键词：
- home（首页）
- photo/album（相册）
- book/guide（指南）
- training/music（训练）
- diary/note（日记）
