# 🦜 无影云电脑 APK 构建配置指南

> 适用于：阿里云无影云电脑（Ubuntu 系统）

## 📋 配置前检查

### 系统要求

| 项目 | 要求 | 检查命令 |
|------|------|---------|
| 操作系统 | Ubuntu 20.04+ | `lsb_release -a` |
| 磁盘空间 | 至少 10GB | `df -h` |
| 内存 | 至少 4GB | `free -h` |
| sudo 权限 | 需要 | `sudo -v` |

### 快速检查

```bash
# 检查系统版本
lsb_release -a

# 检查磁盘空间
df -h /

# 检查内存
free -h

# 测试 sudo 权限
sudo -v
```

### 步骤 1：下载配置脚本

```bash
cd ~
```

- ✅ 安装必要的 SDK 组件

```bash

# 检查 Android SDK
sdkmanager --version

# 检查环境变量
echo $ANDROID_HOME

```

## 📝 手动配置步骤（如果脚本失败）

### 1. 更新系统包

```bash
sudo apt update
sudo apt upgrade -y
```

### 2. 安装 Java 17

```bash
# 安装 OpenJDK 17
sudo apt install -y openjdk-17-jdk

# 验证安装
java -version
javac -version

# 设置 JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64' >> ~/.bashrc
echo 'export PATH=$JAVA_HOME/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

### 3. 安装 Android SDK

```bash
# 创建 SDK 目录
export ANDROID_HOME=~/android-sdk
mkdir -p $ANDROID_HOME
cd $ANDROID_HOME

# 下载 Android 命令行工具
wget https://dl.google.com/android/repository/commandlinetools-linux-9477386_latest.zip -O cmdline-tools.zip

# 解压
unzip cmdline-tools.zip
mkdir -p cmdline-tools/latest
mv cmdline-tools/bin cmdline-tools/lib cmdline-tools/NOTICE.txt cmdline-tools/source.properties cmdline-tools/latest/ 2>/dev/null || true

# 清理
rm cmdline-tools.zip

# 设置环境变量
echo 'export ANDROID_HOME=~/android-sdk' >> ~/.bashrc
echo 'export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin' >> ~/.bashrc
echo 'export PATH=$PATH:$ANDROID_HOME/platform-tools' >> ~/.bashrc
source ~/.bashrc
```

### 4. 安装 SDK 组件

```bash
# 接受许可证
yes | sdkmanager --licenses

# 安装必要的组件
sdkmanager "platform-tools"
sdkmanager "platforms;android-33"
sdkmanager "build-tools;33.0.2"
```

### 5. 验证安装

```bash
# 检查 sdkmanager
sdkmanager --version

# 检查已安装的组件
sdkmanager --list_installed
```

## 🦜 构建鹦鹉成长日记 APK

### 1. 克隆项目（如果还没克隆）

```bash
cd ~
git clone https://github.com/topedu2008/parrot-diary.git
cd parrot-diary
```

### 2. 安装 Node.js 依赖

```bash
npm install
```

### 3. 同步 Capacitor

```bash
npx cap sync android
```

### 4. 构建 APK

```bash
cd android
./gradlew assembleDebug --no-daemon
```

### 5. 查找 APK 文件

```bash
# APK 位置
ls -lh app/build/outputs/apk/debug/app-debug.apk

# 完整路径
readlink -f app/build/outputs/apk/debug/app-debug.apk
```

```

/home/admin/parrot-diary/android/app/build/outputs/apk/debug/app-debug.apk
```

## 🔧 常见问题解决

### Q1: Java 安装失败

**错误信息：** `Unable to locate package openjdk-17-jdk`

**解决方案：**
```bash
# 添加 OpenJDK PPA
sudo add-apt-repository ppa:openjdk-r/ppa
sudo apt update
sudo apt install -y openjdk-17-jdk
```

### Q2: SDK 下载慢

**解决方案：使用国内镜像**

```bash
# 使用清华大学镜像
export ANDROID_SDK_MIRROR=https://mirrors.tuna.tsinghua.edu.cn/android/repository

# 手动下载
wget $ANDROID_SDK_MIRROR/commandlinetools-linux-9477386_latest.zip -O cmdline-tools.zip
```

### Q3: Gradle 构建失败

**错误信息：** `Could not resolve all files`

**解决方案：配置国内 Gradle 镜像**

修改 `android/build.gradle`：
```gradle
repositories {
    google()
    mavenCentral()
    maven { url 'https://maven.aliyun.com/repository/google' }
    maven { url 'https://maven.aliyun.com/repository/public' }
}
```

### Q4: 磁盘空间不足

**解决方案：清理空间**

```bash
# 检查磁盘使用
df -h

# 清理 apt 缓存
sudo apt clean
sudo apt autoremove

# 清理 Node 缓存
npm cache clean --force

# 查找大文件
du -ah ~ | sort -rh | head -20
```

### Q5: 内存不足

**解决方案：调整 Gradle 内存**

修改 `android/gradle.properties`：
```properties
org.gradle.jvmargs=-Xmx1024m -XX:MaxMetaspaceSize=512m
```
