## 📝 第一步：注册微信小程序账号

### 1. 访问注册页面

**网址**: https://mp.weixin.qq.com/

点击「立即注册」

### 2. 选择账号类型

选择 **「小程序」**

### 3. 填写账号信息

| 字段 | 填写内容 |
|------|---------|
| 邮箱 | 你的邮箱（未注册过微信开放平台） |
| 密码 | 设置登录密码 |
| 验证码 | 邮箱验证码 |

### 4. 信息登记

**主体类型选择**：

| 类型 | 费用 | 适合 |
|------|------|------|
| **个人** | 免费 | 个人开发者，功能受限 |
| **企业** | 300 元/年 | 公司/个体户，功能完整 |

**建议选择**：如果有营业执照，选择**个体工商户**（费用低，功能全）

### 5. 填写主体信息

**个人主体**：
- 姓名
- 身份证号
- 手机号

**企业主体**：
- 企业名称
- 统一社会信用代码
- 对公账户信息

### 6. 等待审核

- **个人**: 通常 1-2 个工作日
- **企业**: 通常 3-5 个工作日

审核结果会通过邮件和短信通知

## 🎨 第二步：准备小程序素材

### 1. 小程序图标（Logo）

**要求**：
- 尺寸：**144x144 像素**
- 格式：**PNG**
- 大小：**< 2MB**
- 背景：建议透明或纯色

**你的 Logo**：
- 已有玄凤鹦鹉剪影 logo
- 需要调整到 144x144 像素
- 保存为 `icon.png`

### 2. 小程序启动图

**要求**：
- 尺寸：**750x1334 像素**（建议）
- 格式：**PNG**
- 大小：**< 2MB**

**设计建议**：
- 使用品牌 logo
- 简洁大方
- 符合小程序风格

### 3. 小程序简介

**要求**：
- 字数：**120 字以内**
- 内容：清晰描述小程序功能

**推荐文案**：
```
鹦鹉成长日记是一款专为鹦鹉主人设计的记录工具。提供鹦鹉相册管理、音频训练、饲养指南、成长日记等功能，帮助你科学饲养鹦鹉，记录成长点滴，留下美好回忆。支持本地存储，保护隐私安全。
```

### 4. 服务类目

**选择**：
- 主类目：**工具 - 效率**
- 次类目：**工具 - 查询**

**注意**：某些类目需要提供资质证明

---

## 📤 第三步：上传小程序代码

### 1. 在微信开发者工具中上传

**步骤**：

1. **打开微信开发者工具**
2. **导入项目**（如果还没导入）
   - 项目目录：`miniprogram/`
   - AppID：填写你的小程序 AppID
3. **点击「上传」按钮**
4. **填写版本信息**
   - 版本号：`1.0.0`
   - 项目备注：`初始版本 - 鹦鹉成长日记`
5. **确认上传**

### 2. 登录小程序后台

**网址**: https://mp.weixin.qq.com/

使用注册的账号登录

### 3. 版本管理

**路径**: 管理 → 版本管理

你会看到刚上传的版本：
- **开发版**: 刚上传的版本
- **体验版**: 可以设置体验成员
- **审核版**: 提交审核后
- **线上版**: 发布后的版本

## 📋 第四步：提交审核

### 1. 填写小程序信息

**路径**: 设置 → 基本设置

**需要填写**：

| 项目 | 要求 | 示例 |
|------|------|------|
| **小程序名称** | 4-30 个字符 | 鹦鹉成长日记 |
| **小程序简介** | 120 字以内 | 见上方推荐文案 |
| **小程序图标** | 144x144 PNG | logo.png |
| **小程序封面图** | 900x383 PNG | 可选 |
| **服务类目** | 选择对应类目 | 工具 - 效率 |

### 2. 填写功能页面

**路径**: 设置 → 基本设置 → 功能页面

**需要填写**：

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | pages/index/index | 鹦鹉卡片、快捷操作 |
| 相册 | pages/album/album | 照片/视频管理 |
| 指南 | pages/guide/guide | 饲养知识 |
| 训练 | pages/training/training | 音频训练 |
| 我的 | pages/profile/profile | 个人中心 |

### 3. 隐私保护指引

**路径**: 设置 → 基本设置 → 隐私保护指引

**需要声明**：

- 收集的信息类型（如用户信息、照片等）
- 信息使用目的
- 信息存储方式（本地存储）
- 用户权利（查看、删除、导出等）

**模板**：
```
鹦鹉成长日记隐私保护指引

1. 信息收集
   - 用户信息：昵称、头像（可选）
   - 照片/视频：用户主动上传的鹦鹉照片
   - 录音：用户主动录制的训练音频

2. 信息使用
   - 所有数据仅存储在用户设备本地
   - 不会上传到服务器
   - 不会分享给第三方

3. 用户权利
   - 可随时查看、编辑、删除数据
   - 可导出数据备份
   - 可清除所有数据

4. 联系方式
   - 邮箱：your-email@example.com
```

### 4. 用户协议

**需要创建用户协议文档**

**模板**：
```
鹦鹉成长日记用户协议

1. 服务说明
   鹦鹉成长日记是一款专为鹦鹉主人设计的记录工具。

2. 用户权利
   - 免费使用基本功能
   - 数据本地存储，用户完全控制

3. 用户义务
   - 不得上传违法违规内容
   - 不得用于商业用途

4. 免责声明
   - 数据存储在用户设备，请自行备份
   - 因用户设备问题导致的数据丢失，我们不承担责任

5. 联系方式
   - 邮箱：your-email@example.com
```

**上传方式**：
- 将协议内容放在公众号文章或静态页面
- 在小程序后台填写链接

### 5. 提交审核

**路径**: 版本管理 → 开发版 → 提交审核

**需要填写**：

| 项目 | 说明 |
|------|------|
| **功能页面** | 已填写 |
| **测试账号** | 如需要登录，提供测试账号 |
| **补充说明** | 可选，说明特殊功能 |

**点击「提交审核」**

## ⏰ 第五步：等待审核

### 审核时间

- **通常**: 1-3 个工作日
- **快的话**: 几小时
- **慢的话**: 5-7 个工作日

### 审核状态

可在小程序后台查看：
- **审核中**: 等待审核
- **审核通过**: 可以发布
- **审核失败**: 根据反馈修改后重新提交

### 常见审核失败原因

| 原因 | 解决方案 |
|------|---------|
| 功能不完整 | 确保所有功能可用 |
| 类目选择错误 | 选择正确的服务类目 |
| 隐私政策缺失 | 完善隐私保护指引 |
| 存在 bug | 修复后重新提交 |
| 名称侵权 | 修改名称，避免侵权 |

## 📞 常见问题

### Q1: 个人主体能上线吗？

**A**: 可以，但功能受限。建议选择「工具 - 效率」类目。

### Q2: 审核需要多久？

**A**: 通常 1-3 个工作日，快的话几小时。

### Q3: 审核失败怎么办？

**A**: 根据审核反馈修改，然后重新提交。

### Q4: 上线后可以修改吗？

**A**: 可以，上传新版本，重新提交审核。

### Q5: 小程序需要备案吗？

**A**: 小程序不需要 ICP 备案，但需要微信审核。
