### 特殊信号音详细说明

#### SIG001 🍽️ 喂食信号音

**训练步骤**：
```
第 1 阶段（第 1-3 天）：
1. 准备食物时播放 SIG001
2. 立即把食物放入笼子
3. 鹦鹉进食时轻声表扬
4. 每天 3 次，固定时间

第 2 阶段（第 4-7 天）：
1. 播放 SIG001 后再准备食物
2. 观察鹦鹉是否有期待反应
3. 继续强化关联

第 3 阶段（第 2 周起）：
1. 在远处播放 SIG001
2. 观察鹦鹉是否朝食盆方向移动
3. 成功后给予奖励
```

**预期效果**：鹦鹉听到信号音会立即前往食盆等待

**建议声音**：清脆的铃声、口哨声、或主人说"吃饭啦"

## 🎙️ 音频录制规范

### 技术要求

| 参数 | 标准 | 说明 |
|------|------|------|
| **格式** | MP3/M4A | 兼容小程序播放 |
| **采样率** | 44.1kHz | 标准音频质量 |
| **比特率** | 128kbps 以上 | 保证清晰度 |
| **声道** | 单声道 | 节省空间 |
| **文件大小** | < 5MB/首 | 便于加载 |

### 录制环境

- ✅ 安静房间，关闭空调/风扇
- ✅ 使用外置麦克风（如有）
- ✅ 手机录音时保持距离 20-30cm
- ✅ 避免回声（有家具的房间更好）
- ❌ 避免背景噪音
- ❌ 避免突然的声响

### 人声录制技巧

| 要点 | 说明 |
|------|------|
| **音调** | 稍高音调（鹦鹉更易学习） |
| **语速** | 放慢，清晰发音 |
| **情绪** | 积极、热情 |
| **重复** | 每句重复 2-3 次 |
| **间隔** | 句间停顿 2-3 秒 |

## 🔄 用户音频上传规范

### 上传要求

```markdown
**必填信息**：
- 音频文件（MP3/M4A，<10MB）
- 音频名称（20 字以内）
- 所属分类（6 大类选择）
- 难度等级（⭐/⭐⭐/⭐⭐⭐）

**选填信息**：
- 适用品种（可多选）
- 训练方法说明
- 预期效果描述
- 录制设备/环境说明

**审核标准**：
- ✅ 音质清晰，无明显噪音
- ✅ 内容健康，无不良信息
- ✅ 分类准确，与内容匹配
- ✅ 时长合理（5 秒 -5 分钟）
- ❌ 禁止版权音乐（商业用途需授权）
```

| 行为 | 奖励 |
