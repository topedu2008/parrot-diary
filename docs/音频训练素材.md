## 🎤 唱歌训练

### 训练原理

鹦鹉学习唱歌的关键期是 **6 个月 -2 岁**，这个阶段学习能力最强。

| 因素 | 说明 |
|------|------|
| **最佳时间** | 清晨和傍晚（鹦鹉最活跃） |
| **单次时长** | 5-15 分钟 |
| **频率** | 每天 2-3 次 |
| **周期** | 2-8 周见效（因鸟而异） |

### 唱歌训练方法

**步骤 1：环境准备**
- 安静、无干扰的环境
- 鹦鹉状态放松（不在进食、睡觉）
- 关闭电视、音乐等其他声音

**步骤 2：播放音频**
- 音量适中（不要太大声）
- 单曲循环播放 5-10 分钟
- 主人在旁边陪伴，偶尔跟唱

**步骤 3：正向强化**
- 鹦鹉发出类似声音时立即奖励
- 用兴奋的语气表扬
- 给予零食奖励

**步骤 4：录音对比**
- 每次训练后录音
- 记录训练日期和时长
- 定期回放对比进步

**注意事项**：
- ❌ 不要一次性教太多曲子
- ❌ 不要在鹦鹉疲惫时训练
- ❌ 不要强迫学习
- ✅ 保持耐心，每只鹦鹉学习速度不同

### 技巧训练方法

**训练原则**：
1. **单一指令** - 每次只教一个指令
2. **重复强化** - 每天多次重复
3. **情境关联** - 在合适情境下使用指令
4. **及时奖励** - 正确反应立即奖励

**示例：训练"你好"**

```
第 1 周：
- 每天早晨见面时说"你好"
- 播放音频 T001，每次 5 分钟
- 鹦鹉有任何发声尝试就奖励

第 2-3 周：
- 继续重复，观察是否有模仿迹象
- 录音记录每次训练
- 逐步减少音频播放，增加真人互动

第 4 周及以后：
- 巩固练习
- 在不同情境下使用"你好"
- 扩展到其他指令
```

### 行为训练方法

**上手训练（B001）详细步骤**：

```
准备：
- 鹦鹉站在笼内站棍上
- 准备它喜欢的零食
- 保持环境安静

步骤 1（第 1-3 天）：
1. 慢慢将手指伸向鹦鹉胸前
2. 同时说"上来"
3. 轻推鹦鹉胸部让它失去平衡
4. 鹦鹉站上手指后立即说"真棒"并奖励
5. 每天重复 10 次

步骤 2（第 4-7 天）：
1. 手指伸到鹦鹉面前
2. 说"上来"
3. 等待鹦鹉主动站上来
4. 成功后奖励
5. 逐渐增加停留时间

步骤 3（第 2 周起）：
1. 在不同位置练习
2. 从笼子外开始
3. 增加距离
4. 巩固训练
```

**回笼训练（B002）详细步骤**：

```
准备：
- 鹦鹉站在你手上或肩上
- 笼子门打开
- 准备零食奖励

步骤：
1. 将手慢慢移向笼子
2. 说"回去"
3. 鹦鹉进入笼子后立即奖励
4. 逐步增加你和笼子的距离
5. 最终实现远距离召回

注意：
- 永远不要让回笼变成惩罚
- 回笼后一定要有好事发生（零食、表扬）
```

## 📋 音频文件制作建议

### 录制要求

| 参数 | 要求 |
|------|------|
| **格式** | MP3 或 M4A |
| **采样率** | 44.1kHz |
| **比特率** | 128kbps 以上 |
| **时长** | 10 秒 -2 分钟 |
| **音量** | 适中，避免爆音 |
| **背景音** | 安静，无明显噪音 |

### 录制建议

1. **人声录制**：
   - 使用清晰、高音调的声音（鹦鹉更容易学习）
   - 语速放慢，发音清晰
   - 女性声音通常效果更好

2. **口哨录制**：
   - 音调稳定
   - 旋律简单重复
   - 避免复杂变调

3. **环境音录制**：
   - 使用专业录音设备
   - 避免人声干扰
   - 保持自然感
