## ⚡ 一键构建（推荐）

```bash

cd ~

# 2. 执行配置（自动完成所有步骤）

# 3. 等待完成（约 30-40 分钟）
# APK 会自动构建完成！
```

## 🔧 分步构建

### 步骤 1：安装 Java 17
```bash
sudo apt update
sudo apt install -y openjdk-17-jdk

# 验证
java -version  # 应该显示 "17"
```

### 步骤 2：安装 Android SDK
```bash
# 设置环境变量
export ANDROID_HOME=~/android-sdk
mkdir -p $ANDROID_HOME
cd $ANDROID_HOME

# 下载命令行工具
wget https://dl.google.com/android/repository/commandlinetools-linux-9477386_latest.zip
unzip commandlinetools-linux-9477386_latest.zip
mkdir -p cmdline-tools/latest
mv cmdline-tools/bin cmdline-tools/lib cmdline-tools/NOTICE.txt cmdline-tools/source.properties cmdline-tools/latest/

# 设置 PATH
echo 'export ANDROID_HOME=~/android-sdk' >> ~/.bashrc
echo 'export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin:$ANDROID_HOME/platform-tools' >> ~/.bashrc
source ~/.bashrc

# 安装组件
yes | sdkmanager --licenses
sdkmanager "platform-tools" "platforms;android-33" "build-tools;33.0.2"
```

### 步骤 3：构建 APK
```bash
cd ~/parrot-diary
npm install
npx cap sync android
cd android
./gradlew assembleDebug --no-daemon

# APK 位置
ls -lh app/build/outputs/apk/debug/app-debug.apk
```

## 📱 传输 APK 到手机

### 方法 1：阿里云盘（推荐）
```
1. 浏览器访问 https://www.aliyundrive.com
2. 上传 APK 文件（~/parrot-diary/android/app/build/outputs/apk/debug/app-debug.apk）
3. 手机下载阿里云盘 APP
4. 下载并安装
```

### 方法 2：微信文件助手
```
1. 浏览器访问 https://filehelper.weixin.qq.com
2. 上传 APK 到文件助手
3. 手机微信接收
4. 点击安装
```

### 方法 3：直接复制（如果支持 USB 重定向）
```bash
cp ~/parrot-diary/android/app/build/outputs/apk/debug/app-debug.apk /media/usb/
```

## 🛠️ 常见错误解决

### 错误 1：Java 版本不对
```
错误：Android Gradle plugin requires Java 17
解决：
  sudo apt install -y openjdk-17-jdk
  export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
```

### 错误 2：SDK 未找到
```
错误：sdkmanager: command not found
解决：
  source ~/.bashrc
  echo $ANDROID_HOME  # 应该显示 ~/android-sdk
```

### 错误 3：Gradle 构建失败
```
错误：Could not resolve all files
解决：
  cd ~/parrot-diary/android
  ./gradlew clean assembleDebug --no-daemon
```

### 错误 4：磁盘空间不足
```
解决：
  df -h  # 检查空间
  sudo apt clean
  sudo apt autoremove
```
