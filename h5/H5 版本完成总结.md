## 🚀 立即使用

### 方式 1：本地预览

**服务器已启动：**
```
http://localhost:8080
```

在浏览器打开即可预览！

### 方式 2：部署到 Vercel

```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login

# 部署
cd /home/admin/openclaw/workspace/parrot-diary/h5
vercel --prod
```

**部署成功后获得链接：**
```
https://parrot-diary.vercel.app
```

### 方式 3：部署到阿里云 OSS

1. 登录阿里云 OSS 控制台
2. 创建 Bucket（选择公共读）
3. 上传 `h5/` 目录所有文件
4. 开启静态页面托管
5. 获得访问链接

---

## 📱 添加到手机主屏幕

### iOS (Safari)

```
1. Safari 打开链接
2. 点击底部「分享」按钮
3. 选择「添加到主屏幕」
4. 点击「添加」
```

### Android (Chrome)

```
1. Chrome 打开链接
2. 点击右上角「⋮」菜单
3. 选择「添加到主屏幕」
4. 点击「添加」
```

**添加后桌面会出现 APP 图标：🦜 鹦鹉成长日记**
